# Pokedex
Uma página HTML que carrega uma lista simples de pokémons, os dados utilizados vêm da API pública [PokeAPI]('https://pokeapi.co/').

Este projeto foi criado com a itenção de aplicar conceitos de javascript e consumo de API's REStful. E claro, entender melhor o processo para a criação de páginas dinâmicas.

*Este projeto foi criado com o auxílio do Software Engineer Lead: **Renan Johannsen***. Com a adição de alguns detalhes por mim mesmo, em breve haverá a página de detalhes do pokémon.
